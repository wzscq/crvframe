# crvframe
A lowcode framework for CRUD applications. You simply create configuration files in JSON format to build your application. 

# Components
* *mainframe*: A front-end main component of the framework, It contains the common UI functions of a application, login page„ÄÅapplication menus etc. It uses iframe to load other front-end components or custom pages. It provides a group of abstraction operation API for other front-end pages to interact with back-end services.

* *listview*: A front-end component of the framework used to retrieve data and show the data in table.
